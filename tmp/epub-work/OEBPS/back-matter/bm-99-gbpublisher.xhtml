Esta revista fue generada (24/06/2026) a partir de un flujo de trabajo reproducible utilizando gbpublisher, una plataforma cliente-servidor de código abierto diseñada para equipos editoriales académicos de América Latina.

Permite convertir manuscritos en publicaciones compatibles con los principales indexadores internacionales, siguiendo estándares como XML-JATS y manteniendo la filosofía de transparencia en todo el proceso editorial.

Proyecto disponible en: https://github.com/albertomoyano/gbpublisher
